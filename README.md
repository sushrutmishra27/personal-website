# Sushrut KM — Personal Website

A static personal website for Sushrut KM, a developer turned marketer.
Developer Content Marketer at Bito.ai and Head of Growth Content at Devable.ai.

## Stack

- Plain HTML, CSS, and vanilla JavaScript (no build step)
- Monochrome design (black / white / grey) with a light/dark/auto theme toggle
- Day- and time-aware headline based on the host's local timezone

## Structure

- `index.html` — home
- `about.html` — about
- `projects.html` — projects
- `kind-words.html` — testimonials
- `writing.html` — writing
- `toolkit.html` — toolkit
- `reading.html` — reading list
- `case-studies.html` — case studies
- `styles.css` — shared styles
- `script.js` — shared scripts (theme toggle, time-aware headline, etc.)

## Run locally

Open `index.html` in a browser, or serve the folder:

```bash
python3 -m http.server 8000
# then visit http://localhost:8000
```
